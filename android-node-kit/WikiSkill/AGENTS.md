# Android Node Kit Wiki Contract

This wiki tracks the Android edge-device install workflow for Quant-M.

- Treat `raw/` as source truth snapshots from this workspace.
- Treat `wiki/` as maintained summaries, procedures, and decisions.
- Keep manifests and inventory references workspace-relative.
- Mark missing bundle artifacts explicitly instead of implying they exist.
- Record every meaningful update in `wiki/log.md`.
