# Quant-M v0.1.0-beta

Release posture: public beta candidate.

Quant-M v0.1.0-beta is a local-first, CLI-first Rust control plane for governed AI work. It is designed to make agent work inspectable: evidence is written locally, replay is side-effect free, compact packets preserve handoff state, and the context guardian prepares continuity handoffs when local session evidence shows drift risk.

## Beta Meaning

This release is ready for developers and operators who are comfortable with a terminal-first workflow and want to evaluate Quant-M's governance model.

Expect sharp edges:

- no packaged release binaries yet
- no installer script yet
- no hosted dashboard
- limited public onboarding polish
- launchd/systemd autostart docs still developing

## What Works Locally

- `quant-m init`
- `quant-m init-truth`
- `quant-m setup --non-interactive --runtime-profile laptop --context-guardian true`
- `quant-m context-status`
- `quant-m consensus --dry-run "..."`
- `quant-m replay <session_id>`
- `quant-m compact <session_id>`
- `quant-m context guard --json`
- `quant-m cost summary`

## What Quant-M Is Not

- not a hosted agent platform
- not a trading bot
- not a replacement for Codex, Claude Code, or other coding agents
- not production enterprise software yet
- not automatic unchecked agent execution
