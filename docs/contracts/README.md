# Contracts

Use this folder for durable machine-readable onboarding outputs and the human-readable notes that explain how downstream systems should consume them.

Current contract set:

- `ponboarding-to-staff-os-handoff.md`
- `ponboarding-handoff.schema.json`
- `repo-scorecard.schema.json`
- `model-stack-recommendation.schema.json`

Treat these files as the canonical output shape for project onboarding data that will later be consumed by Staff OS or other execution runtimes.
